Olá! Sou o João :)

Profissional de TI com um grande apreço pelo que faço. Sou um autodidata movido por uma curiosidade constante, o que me leva a desenvolver soluções criativas tanto para problemas simples quanto complexos.

Gosto de explorar sistemas em geral — computacionais, elétricos e automotivos. Meu objetivo é sempre entender como cada detalhe funciona, por que funciona, e como pode ser melhorado.

Sou formado em Análise e Desenvolvimento de Sistemas, mas tenho anos de experiência com virtualização, infraestrutura de redes e suporte técnico.

Além da TI, tenho conhecimentos em eletrônica, que uso como hobby e ferramenta para visualizar problemas de forma mais ampla. Acredito que sistemas bem-sucedidos são resultantes de equilíbrio, tendo como pilares a confiabilidade, a simplicidade e a segurança.

Atualmente, estou focado em:

Golang IA & LLMs Cloud Self-Hosted Docker Linux

Projetos

Soluções Self-Hosted


Meu mini servidor próprio atuando como um cluster do Proxmox VE, possibilitando a oferta de inúmeros serviços dentro e fora da minha rede local (incluindo este site!)

Nask AI


Plataforma própria de inteligência artificial, baseando-se em soluções de código aberto para pesquisa e desenvolvimento de LLMs personalizados.

Nask Garage


Aplicação web baseada no LubeLogger para gerenciar inúmeras coisas sobre carros.

Desde cálculos com despesas de manutenção, consumo de combustível e notificação para manutenções futuras.

Este Site


Desenvolvido com base nos princípios de simplicidade, confiabilidade e segurança.

Roda em HTML5 + CSS com o backend desenvolvido em Go em um container Docker.

Segurança de Rede


Rede local protegida com o AdGuard Home, um DNS sinkhole que não somente bloqueia anúncios chatos, mas também protege a conexão contra sites maliciosos e suspeitos.

Também auxilia na privacidade da rede com o uso de criptografia (DNS-over-HTTPS).

VPN


Solução auto-hospedada de VPN para acesso à rede local de qualquer lugar de forma segura.

Graças ao recurso Exit Node do Tailscale, é possível rotear todo o trafégo de rede onde quer que eu esteja para a minha rede de casa.

Arduino EEC


Utilização do Arduino como uma interface para comunicação para o protocolo DCL, usada em ECUs de veículos da Ford até o final dos anos 90.

Ainda em desenvolvimento, o objetivo é obter de dados de diagnóstico e funcionamento do veículo.

Arduino DHT


Utilizo para o hub de automação residencial da minha casa, um sensor de temperatura e umidade DHT11, ligado em um Arduino Mega que repassa as condições do ambiente via interface serial diretamente para a VM do Home Assistant que possuo no cluster do Proxmox.

Essa integração permite automatizar várias funções de climatização, junto aos condicionadores de ar dentro de casa

Chorumebook


Das minhas experiências malucas com o ChromeOS em um Samsung Chromebook 3, modifiquei o firmware do meu Chromebook para uma implementação UEFI de código aberto para permitir a instalação do Windows e distros Linux.

Obtive um laptop que apesar do hardware bastante modesto, proporciona conectividade robusta (Wi-Fi 5) com mais de 12 horas de bateria. Um Thin Client portátil.
